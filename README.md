# Lista em Fila Dinâmica de Números Inteiros.

Um trabalho que fiz da matéria de Estruturas de Dados I do curso de Sistema de Informação da UFPI – Picos –Piauí – Brasil.

É uma lista em fila dinâmica que recebe números inteiros em linguagem C.
 Ele cria a lista em fila para armazenar os números, insere os números na lista dado pelo usuário, busca os números que estão
 dentro da lista, remove um número escolhido pelo usuário por vez na lista, mostra o número no começo da fila que é a lista, mostra o número no final da fila,
 mostra os elementos que estão dentro da lista e exclui a lista liberando-a....
